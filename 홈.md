# 🏠 홈 대시보드


---

## 📅 오늘 일간 노트
```dataview
LIST
FROM "00_Inbox"
WHERE file.name = dateformat(date(today), "yyyy-MM-dd")
```

---

## 📌 진행 중 지시업무
```dataview
TABLE 지시자, 마감일, 중요도
FROM "01_Project"
WHERE 상태 = "진행중"
SORT 마감일 ASC
```

---

## ✅ 미완료 체크리스트
```dataview
TASK
FROM "01_Project" OR "02_Area"
WHERE !completed
SORT file.mtime DESC
LIMIT 20
```

---

## 📋 이번 주 회의록
```dataview
TABLE 참석자, 장소
FROM "01_Project" OR "02_Area"
WHERE 날짜 >= date(today) - dur(7 days)
AND file.name != "회의록_템플릿"
SORT 날짜 DESC
```

---

## 📈 투자 종목 현황
```dataview
TABLE 티커, 섹터, 상태, 분석일
FROM "02_Area/투자분석"
WHERE file.name != "투자분석_템플릿"
SORT 분석일 DESC
```

## 🕐 최근 수정 노트
```dataview
TABLE file.mtime AS "수정일", file.folder AS "위치"
WHERE file.folder != "00_Inbox/Templates"
SORT file.mtime DESC
LIMIT 10
```
